### MVVM 모델에서의  Vue
#### Vue 는 MVVM 패턴의 뷰모델 레이어에 해당하는 화면단 라이브러리
#### ViewModel 레이어인 Vue 에서는 DOM 에서 오는 이벤트를 DIM Listeners 를 통해 이벤트를 확인하고 Model 에 전달
#### 또한 Model 에서 결과인 데이터를 ViewModel 레이언인 Vue 가 받아 DOM 에게 전달 한다.
